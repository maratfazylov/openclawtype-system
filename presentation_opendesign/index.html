Deep Agents Workshop01 Контекст
ментальная модель

Эволюция проста: от ответа к действию

Участники увидят, где заканчивается чатбот, почему обычный tool-calling не дает полноценного цикла и что именно добавляет deep agent.

Чатбот / RAG

Отвечает на вопросы и читает базу знаний. Ограничение: не действует.

Tool-calling

Вызывает API и пишет в БД. Ограничение: нет устойчивого цикла.

Deep Agent

Планирует сам, итерирует, управляет браузером и останавливается перед опасным действием.

Заметка докладчика: показать лестницу зрелости без маркетинга: участнику должно стать ясно, в какой момент нужен именно deep agent.

Deep Agents Workshop08 Stack
инструменты без vendor lock-in

Стек выбран так, чтобы агент был заменяемым и наблюдаемым

Оркестрация
  • LangGraph: StateGraph, ToolNode, checkpointer, interrupt
  • LangChain: базовые абстракции и `@tool`
Исполнение
  • OpenRouter: claude / gpt / gemini / qwen через единый API
  • Playwright: headless browser как инструмент LLM
Контроль
  • LangSmith: трассы, токены, latency, дебаг
  • SQLiteSaver: состояние между вызовами
Заметка докладчика: подчеркнуть причину OpenRouter — одна строка меняет модель, это удобно под разные корпоративные политики.
